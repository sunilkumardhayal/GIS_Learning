Watershed Delineation in ArcGIS Pro

This guide provides the complete workflow for delineating a watershed using a Digital Elevation Model (DEM) in ArcGIS Pro. The steps cover the core process and optional advanced analyses.

Step 1: Add Digital Elevation Model (DEM) Data

The first step is to add your DEM data (e.g., a GeoTIFF file) to ArcGIS Pro by dragging it into the map view. You can adjust the layer's symbology to better visualize the terrain.

Step 2: Fill Sinks

This process corrects imperfections in the DEM to create a hydrologically correct surface, ensuring continuous water flow for the analysis.

Tool: Fill

Location: Geoprocessing > Toolboxes > Spatial Analyst Tools > Hydrology > Fill

Step 3: Create Flow Direction Raster

This tool creates a raster that shows the downslope direction water will flow from each cell.

Tool: Flow Direction

Location: Geoprocessing > Toolboxes > Spatial Analyst Tools > Hydrology > Flow Direction

Step 4: Create Flow Accumulation Raster

This tool calculates the number of upstream cells flowing into each cell, which helps identify and visualize stream channels.

Tool: Flow Accumulation

Location: Geoprocessing > Toolboxes > Spatial Analyst Tools > Hydrology > Flow Accumulation

Step 5: Create Outlet Points (Pour Points)

You must define the specific outlet point for your watershed. This involves two actions:

Action: Create a new point feature class.

Location: In the Catalog pane, right-click your folder and go to New > Shapefile (or Feature Class within a geodatabase).

Action: Add a point to the new layer at your desired outlet location (on a high-flow-accumulation cell).

Location: Use the tools in the Edit tab, specifically the Create tool. Remember to save your edits.

Step 6: Delineate the Watershed

This is the final step that generates the watershed boundary based on your flow direction raster and outlet point.

Tool: Watershed

Location: Geoprocessing > Toolboxes > Spatial Analyst Tools > Hydrology > Watershed

Step 7: Convert Raster to Polygon (Optional)

To make the watershed raster more usable for area calculations and other vector analyses, convert it to a polygon feature.

Tool: Raster to Polygon

Location: Geoprocessing > Toolboxes > Conversion Tools > From Raster > Raster to Polygon

Optional Advanced Steps

Generate River Lines

You can create a vector line feature representing the river network within your watershed.

1. Tool: Reclassify

Description: First, isolate cells with high flow accumulation values from your Flow Accumulation raster.

Location: Geoprocessing > Toolboxes > Spatial Analyst Tools > Reclass > Reclassify

2. Tool: Raster to Polyline

Description: Next, convert the reclassified raster (representing streams) into a line feature.

Location: Geoprocessing > Toolboxes > Conversion Tools > From Raster > Raster to Polyline

3. Tool: Clip

Description: Finally, you can clip the generated river lines to your watershed boundary for a clean map.

Location: Geoprocessing > Toolboxes > Analysis Tools > Extract > Clip
